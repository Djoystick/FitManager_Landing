# Prompt for Claude IDE: Полный редизайн FitManager_SITE

**Роль:** Ты — Senior Frontend Engineer и UI/UX Архитектор мирового уровня. Твоя специализация — создание премиальных, футуристичных и сверхплотных интерфейсов (dashboard-style) с вау-эффектом.

**Задача:** Сделать полный редизайн и рефакторинг проекта `FItManager_SITE` (лендинг игры FitManager). 
Текущий сайт выглядит слишком пусто, разрозненно и не передает атмосферу киберпанк/Web3 проекта. 

Требуется:
1. Изучить файл `DESIGN.md` (находится в корне сайта) — это основа нашей дизайн-системы (Midnight Command Center).
2. Изучить исходники игры из соседнего репозитория `fitmanager_game` (в частности `fitmanager_game/docs/FitManager_PitchDeck.md` или `README.md`), чтобы вытащить **всю возможную информацию** о лоре, фичах, токеномике, механике игры и инфраструктуре клуба.
3. Полностью переписать структуру лендинга. Сделать его максимально информативным, компактным (без огромных белых/черных пустот) и визуально богатым.
4. Добавить больше секций (Roadmap, Tokenomics, Lore, Gameplay Mechanics) используя данные из проекта игры.
5. Использовать стекломорфизм (frosted glass), неоновые градиенты, микро-анимации (framer-motion) и сложные тени (inset shadows), описанные в `DESIGN.md`.
6. Сохранить текущую реализацию мультиязычности (i18n) и поддержку светлой/темной темы, но интегрировать их в новый плотный дизайн.

**Технические ограничения:**
- Стек: Next.js (App Router), Tailwind CSS, Framer Motion, Lucide React.
- Не использовать сторонние библиотеки UI-компонентов (MUI, Chakra и т.д.), все компоненты строить на основе Tailwind и кастомных CSS переменных.
- Дизайн должен быть идеально адаптивным под мобильные устройства (Telegram Mini App feel).

**Критерий успеха:**
Пользователь должен зайти на сайт и сказать "ВАУ". Интерфейс должен выглядеть как терминал космического корабля или продвинутая крипто-биржа — плотный поток полезной информации, упакованный в потрясающий UI.

**План действий:**
1. Проанализируй данные.
2. Предложи новую архитектуру страниц (компоненты).
3. Напиши код шаг за шагом.
